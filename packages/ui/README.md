# @zyplux/ui

Shared utilities for the zyplux-ai monorepo.

## Exports

- `@zyplux/ui/lib/utils` - `cn()` for className merging (`clsx` + `tailwind-merge`)

## Usage

```typescript
import { cn } from '@zyplux/ui/lib/utils';
```

## Development

This package exports source TypeScript files directly (no build step). Consuming applications handle transpilation via Vite.
